Salesforce
1
User → Node API POST /api/chat/message
Sends message with session context and auth token
2
Node API → Policy evaluate(message, userRole)
Checks content policy, rate limits, and permissions
3
Policy → Node API PolicyResult { allowed: true }
4
Node API → Memory search(orgId, query, topK=5)
Retrieves relevant context from vector store (pgvector)
5
Memory → Node API contextChunks: List[Memory]
6
Node API → Zentom AI generate(prompt, context, model)
Sends enriched prompt with RAG context to AI engine
7
Zentom AI → Zentom AI streamGenerate() → SSE chunks
Streams response tokens via Server-Sent Events
8
Zentom AI → Node API AIResponse { text, tokens, model }
9
Node API → Memory store(orgId, exchange)
Persists conversation to memory for future context
10
Node API → Salesforce logInteraction(sessionId, metadata)
Creates AuditLog and updates Incident if linked
11
Node API → User 200 OK { response, sessionId }
Returns AI response with streaming support
